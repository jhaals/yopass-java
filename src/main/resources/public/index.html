Yopass Share Secrets Securely

In memory storage

The encrypted message is never stored on disk leaving no traces of your data

Self destruction

All messages have a fixed time to live and will be deleted automatically after expiration

Visible Once

The decrypted message can only be viewed once before automatic deletion

API

This site is just a frontend to the Yopass API, use it to extend other applications. A CLI tool is also available here

AES-256 Encryption

Strong encryption method with no copies of the decryption key

No caching

Caching headers are disabled

Rate limiting

Built in rate limiting preventing brute force attacks

Open Source

Yopass is an Open Source project meaning full transparency and the possibility to submit features, fix bugs or run the software yourself
